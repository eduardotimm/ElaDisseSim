# Ela Disse Sim!

Projeto do site do nosso casamento!

## 🚀 Stack Tecnológica
- **Backend:** C# .NET (Entity Framework Core, Minimal APIs)
- **Arquitetura:** Vertical Slice Architecture
- **Banco de Dados:** SQLite (ou PostgreSQL)
- **Frontend:** React (criado com Vite), TypeScript e Tailwind CSS

## 📌 Funcionalidades

1. **Save the Date:** Página inicial com contagem regressiva para a data do casamento.
2. **Lista de Presentes (Pix):** Sistema de presentes onde os convidados podem gerar um código Pix (QR Code e chave Copia e Cola) para os noivos.
3. **RSVP (Confirmação de Presença):** Sistema organizado por núcleo familiar. O convidado acessa seu convite e pode confirmar a si mesmo e aos seus acompanhantes.
4. **Painel Admin:** Área restrita para gestão do site (alterar data, visualizar confirmações de RSVP e presentes recebidos).
